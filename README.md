# Object Navigation App for Blind Persons

An Android-based assistive navigation system for visually impaired people. 
The app uses **object detection** and **depth estimation** to guide the user safely.

---

## Features
- Real-time object detection
- Depth estimation to avoid obstacles
- Audio feedback for navigation
- Easy-to-use interface for visually impaired

---

## How to Run the Project
1. Download the project from GitHub as a ZIP file or clone it to your computer.
2. Open the project in **Android Studio**.
3. Build the project.
4. Run the app on an Android device or emulator.
